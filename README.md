🔢 Handwritten Digit Classifier: Deep Learning with CNN & StreamlitAn interactive web application that uses a Convolutional Neural Network (CNN) to recognize and classify handwritten digits (0-9). This project demonstrates the full machine learning lifecycle—from data preprocessing and model architecture design to real-time deployment.ShutterstockExplore🚀 Key FeaturesReal-time Inference: Users can upload their own handwritten digit images for instant classification.Automated Preprocessing Pipeline: Integrated Pillow logic to convert, invert, and resize images to the standard $28 \times 28$ grayscale format.Performance Optimization: Utilizes @st.cache_resource to ensure the heavy TensorFlow model is only loaded into memory once.Probability Visualization: Displays the AI's confidence level for each prediction.🛠️ Technical StackProgramming: Python 3.10+Deep Learning: TensorFlow / KerasFrontend/UI: StreamlitMathematical Operations: NumPyImage Processing: Pillow (PIL)Dataset: MNIST (Modified National Institute of Standards and Technology)🧠 Model ArchitectureThe project utilizes a multi-layer CNN designed for spatial feature extraction:Convolutional Layer (Conv2D): 32 filters to detect edges and patterns.Max Pooling: Reduces spatial dimensions to focus on the most critical features.Flatten Layer: Converts 2D feature maps into a 1D vector for classification.Dense Layers: Fully connected layers with ReLU activation for non-linearity.Output Layer: Uses Softmax activation to provide a probability distribution across 10 classes (0-9).📂 Project StructurePlaintext├── app.py               # Streamlit application & Frontend logic
